## Installation

`.tmux.conf`

```
# tmux plugin manager
# https://github.com/tmux-plugins/tpm/
# List of plugins
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
set -g @plugin 'tmux-plugins/tmux-urlview'
set -g @plugin 'mistweaverco/tmux-sexy'
```
